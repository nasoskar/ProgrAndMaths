This project is part of the "Programming and Maths" course and contains 2 folders: Task 1, and Task 2.

**Task 1**:<br>
**test.ipynb**: the notebook containing all the code of task 1.<br>
**requirements.txt**: necessary libaries required to run the notebook.<br>
**winequalityN.csv**: the csv file with the dataset.<br>
**latest_finetuned_results.csv**: a csv file with the parameters and performance of all configurations that ran during tuning the model.<br>

**Task 2**:<br>
**ecg_project.ipynb**: the notebook containing all the code of task 2.<br>
**ECG200_TEST.txt**: the test set of the dataset.<br>
**ECG200_TRAIN.txt**: the train set of the dataset.<br>
**hyperparameter_search_results.csv**: a csv file with the parameters and performance of all configurations that ran during tuning the model.<br>
